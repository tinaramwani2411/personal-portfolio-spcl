ReportMate

ReportMate is a Complaint Management System (CMS) currently in the development phase, designed to provide a reliable and user-friendly platform for submitting and tracking complaints. The application is being developed using Android (Java), XML for UI design, and Firebase for real-time database and authentication. It allows users to securely register complaints, track their report status in real-time, and maintain transparency throughout the process. The system focuses on user trust, smooth navigation, and efficient complaint handling.
